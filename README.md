# dev-workspace - Gestión de Proyectos de Software

Proyecto de TypeScript para la gestión de desarrolladores, pull requests e issues de un proyecto de software, con reportes visuales en consola.

## 📋 Descripción

Este proyecto implementa un sistema de tipos para gestionar:
- **Desarrolladores** con rol, seniority, tecnologías y disponibilidad
- **Pull Requests** con seguimiento de estados y revisores
- **Issues** (bugs y mejoras) con prioridad y asignación
- **Proyectos** agrupando equipos, PRs e issues

## 🚀 Instalación

```bash
npm install
```

## ▶️ Ejecución

```bash
npm run dev
```

## 📦 Scripts disponibles

- `npm run build` - Compilar TypeScript a JavaScript
- `npm run dev` - Compilar y ejecutar
- `npm start` - Ejecutar versión compilada

## 🛠️ Tecnologías

- TypeScript 6.0.2
- Node.js
